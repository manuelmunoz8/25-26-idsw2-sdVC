# Diseñado e implementacion de [Plataforma Interna de Investigación](https://github.com/31Diego/25-26-IdSw1-SdR)

Sistema integral para la gestión y asignación de trabajos de investigación dirigido a docentes e investigadores de FUNIBER. La plataforma facilita la organización de proyectos, la asignación de tareas y fomenta la comunicación colaborativa entre los miembros de los equipos de investigación.

## 🚀 Propósito del Proyecto

Optimizar el flujo de trabajo investigativo mediante una herramienta centralizada que permita:
- **Gestión de Proyectos:** Creación, seguimiento y cierre de iniciativas de investigación.
- **Asignación Eficiente:** Distribución de carga de trabajo entre investigadores y docentes.
- **Colaboración Directa:** Espacios de comunicación integrados para los equipos asignados.
- **Trazabilidad:** Seguimiento detallado de entregables y avances.

## 📂 Estructura del Repositorio

El proyecto se divide en tres pilares fundamentales:

- **`backend/`**: API robusta desarrollada en Node.js (NestJS) que gestiona la lógica de negocio, persistencia de datos y seguridad.
- **`frontend/`**: Interfaz de usuario intuitiva construida con React, diseñada para ofrecer una experiencia fluida tanto a coordinadores como a investigadores.
- **`documents/`**: Documentación técnica exhaustiva, incluyendo análisis de requisitos, diseño de arquitectura y gestión del proyecto.

## 🛠️ Tecnologías Principales

- **Frontend:** React, TypeScript, CSS.
- **Backend:** Node.js, NestJS, TypeScript.
- **Documentación:** PlantUML, Markdown.
