# Contexto Frontend

Este módulo contiene toda la lógica visual y de interacción del sistema.

---

# Responsabilidades

- Componentes visuales
- Navegación
- Gestión de estado frontend
- Validaciones visuales
- Experiencia de usuario

---

# Reglas

- Mantener consistencia visual.
- Priorizar componentes reutilizables.
- Evitar lógica de negocio compleja en UI.
- Documentar decisiones importantes.
- Mantener nombres descriptivos.

---

# Documentación

Actualizar:

/frontEnd/documentacion/ai_log.md

cuando ocurra:

- creación de componentes importantes
- cambios de arquitectura frontend
- modificaciones relevantes de UI
- decisiones técnicas importantes

Actualizar:

/frontEnd/documentacion/decisiones.md

cuando se tome una decisión oficial del módulo.

---

# Formato de Logs

## [HH:MM] Título breve

**Prompt:** solicitud realizada

**Resultado:** implementación o propuesta generada

**Decisión:** decisión tomada y justificación
